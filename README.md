# We.js Mailchimp subscription plugin

Plugin to add route and widget with support for diferent lists in each widget.

## Instalation:
```
we i we-plugin-mailchimp-sb
```

## API

### New subscription
```
post /newsletter/subscribe
  body: {widgetId, email, name}
```
